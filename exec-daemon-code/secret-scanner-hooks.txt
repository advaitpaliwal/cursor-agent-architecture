# Cursor Cloud Agent Git Hooks - Secret Scanning System
# Location: ~/.cursor/agent-hooks/<base64-encoded-workspace-path>/
# Base64 "L3dvcmtzcGFjZQ" = "/workspace"
#
# Hook architecture:
# 1. .dispatcher - Universal hook runner (symlink target)
# 2. pre-commit.cursor - Secret value scanner for staged files
# 3. commit-msg.cursor - Secret value scanner for commit messages
# 4. commit-msg.cursor.co-author - Auto-appends user as co-author
# 5. .cursor-original-hooks-path - Points to /workspace/.git/hooks

## .dispatcher (universal hook runner)
# All git hook names symlink to this script
# Runs original hooks first, then all .cursor* hooks for that type
# Pattern: $HOOKS_DIR/$HOOK_NAME.cursor*

## pre-commit.cursor - Cloud Agent Secrets Scanner
# Prevents committing files containing user-configured secret values
# Env var: CLOUD_AGENT_INJECTED_SECRET_NAMES (comma-separated)
# Each secret name maps to an env var with the actual value
#
# Smart merge handling:
# - If MERGE_HEAD + AUTO_MERGE exist: compares index against AUTO_MERGE
# - Older Git fallback: parses conflict files from MERGE_MSG
# - Binary file detection via git diff --numstat (- - filename)
#
# Skip conditions:
# - Secret value < 8 chars (avoid false positives)
# - Lines with "// pragma: allowlist secret"
# - Binary files checked via diff added lines only
#
# Output: "CURSOR_SECRET_SCAN_BLOCKED=1" to stderr on block
# Exit 1 to block commit

## commit-msg.cursor - Commit Message Secret Scanner
# Same CLOUD_AGENT_INJECTED_SECRET_NAMES env var
# Scans commit message file for secret values
# Same 8-char minimum threshold

## commit-msg.cursor.co-author
# Appends: Co-authored-by: <user name> <user email>
# Only if no Co-authored-by already present
# Current user: Advait Paliwal <advaitspaliwal@gmail.com>

## Key Design Insights:
# 1. Secrets are injected as ENV VARS, not stored in files
# 2. The scanner only checks ADDED lines in diffs (not existing code)
# 3. pragma: allowlist secret can bypass the check per-line
# 4. Binary files are scanned via grep -a on diff added lines
# 5. The co-author hook ensures the human user gets attribution
#    even when the agent makes the commit
