## Key Capabilities
1. **Container Management**: Full CRUD (create/start/delete/pause/resume)
2. **OCI Image Pulling**: Direct registry pulls with auth (Bearer tokens)
3. **Block Mount (blkmnt)**: Custom block device layer for container filesystem
   - XFS formatting for block devices
   - Block upload/download results
   - Userspace block device (ublk) handler
4. **Snapshots**: Create container snapshots with streaming progress
5. **Log Streaming**: Attach to container logs with ServerStreaming
6. **Tar Upload**: Upload tar archives into containers (ClientStreaming)
