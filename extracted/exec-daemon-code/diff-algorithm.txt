1. ARCHITECTURE OVERVIEW
================================================================================

KEY INSIGHT: The exec-daemon does NOT implement the search-and-replace diff
algorithm itself. The architecture splits responsibilities:

- **Server-side (aiserver)**: The LLM outputs tool calls with parameters like
  `old_string`, `new_string`, `contents`, etc. These are sent as
  ClientSideToolV2Call protobuf messages (e.g., EDIT_FILE=7, EDIT_FILE_V2=38,
  APPLY_AGENT_DIFF=50).

- **Client-side (Cursor IDE / exec-daemon)**: For EDIT_FILE/EDIT_FILE_V2,
  the actual text matching and replacement (search-replace) happens in the
  Cursor IDE frontend or the aiserver, NOT in the exec-daemon. The daemon
  receives the final resolved `fileText` and uses `LocalWriteExecutor` to
  write it to disk.

- **ApplyAgentDiff (tool ID 50)**: A higher-level tool that takes an
  `agent_id` and applies all accumulated changes from that agent's session.
  The actual diff resolution happens server-side. The result is a list of
  `AppliedAgentChange` entries (created/modified/deleted) with before/after
  content.

- **exec-daemon's role**: File I/O, permission checking, change tracking,
  line ending normalization, encoding detection, and hooks execution.

The search-and-replace matching logic (old_string -> new_string) with
whitespace-insensitive fallback and fuzzy matching is implemented in the
Cursor IDE extension (TypeScript), not in the daemon bundle.

class LocalWriteExecutor {

    workspacePath;

                { workspacePaths, workspacePath, worktreeMainPath, onWriteForAiTracking }) {

    async execute(parentCtx, args) {
        // args.path:      Target file path
        // args.fileText:  Content to write
        // args.fileBytes: Binary data (if non-empty, skip text processing)
        // args.toolCallId: For tracking
        // args.returnFileContentAfterWrite: bool

        const resolvedPath = resolvePath(path, this.workspacePath);

        // 1. Skip agent tracking for internal paths (agent-tools, swarm-agents)
        const shouldSkipAgentTracking = isCursorProjectsInternalPath(resolvedPath, [
            "agent-tools", "swarm-agents"
        ]);

        // 2. Block worktree paths
        if (shouldBlockWorktreePath({targetPath, workspacePaths, mainWorktreePath})) {
            return WriteResult { case: "error", value: WORKTREE_GUARD_ERROR };
        }

        // 3. Read existing content for change tracking
        let beforeContent;
        try {
            beforeContent = await readText(resolvedPath);
        } catch { /* File doesn't exist */ }

        // 4. Check permissions (may require user approval)
        const shouldBlock = await permissionsService.shouldBlockWrite(ctx, resolvedPath, fileText);
        if (shouldBlock) {
            // Handle: needsApproval -> requestApproval()
            //         userRejected -> WriteRejected
            //         permissionDenied -> WritePermissionDenied
        }

        // 5. Ensure parent directory exists
        await mkdir(dir, { recursive: true });

        // 6. Prepare buffer with line ending normalization
        if (fileBytes !== undefined) {
            buffer = fileBytes;  // Binary mode
            lines = 0;
        } else {
            // Text mode: detect encoding, normalize line endings
            const textResult = await getBufferForWrite(resolvedPath, fileText, workspacePath);
            buffer = textResult.buffer;
            lines = textResult.lines;
        }

        // 7. Write file with fsync
        await writeFile(resolvedPath, buffer, { flush: true });
        // Also sync the directory (best effort)
        dirHandle = await open(dirname(resolvedPath), O_RDONLY);
        await dirHandle.sync();

        // 8. Track change in FileChangeTracker
        if (fileChangeTracker && fileBytes === undefined && !shouldSkipAgentTracking) {
            await fileChangeTracker.trackChange(resolvedPath, beforeContent, fileText, metadata);
        }

        // 9. Call AI code tracking for cursor-blame (best-effort)
        if (onWriteForAiTracking && fileBytes === undefined && !shouldSkipAgentTracking) {
            await onWriteForAiTracking({
                path: resolvedPath,
                beforeContent,
                afterContent: fileText,
                metadata: { toolCallId },
            });
        }

        // 10. Optionally re-read file content after write
        if (args.returnFileContentAfterWrite) {
            fileContentAfterWrite = await readText(resolvedPath);
        }

        return WriteResult {
            case: "success",
            value: WriteSuccess { path, linesCreated, fileSize, fileContentAfterWrite }
        };
    }
}

class FileChangeTracker {
    workspacePath;
    changes = new Map();    // Map<path, { path, before, after, metadata }>
    listeners = new Set();  // Set<Function> for change notifications

    constructor(workspacePath) {
        this.workspacePath = workspacePath;
    }

    // Subscribe to change notifications. Returns unsubscribe function.
    subscribe(listener) {
        this.listeners.add(listener);
        return () => this.listeners.delete(listener);
    }

    notify(change) {
        for (const listener of this.listeners) listener(change);
    }

    // Core tracking method
    async trackChange(path, before, after, metadata) {
        const existingChange = this.changes.get(path);
        if (existingChange) {
            // MERGE: Keep original "before" state (from first change)
            const originalBefore = existingChange.before;
            const mergedMetadata = metadata || existingChange.metadata;

            // Check for NO NET CHANGE
            if (originalBefore === after ||
                (originalBefore === undefined && after === undefined)) {
                // Net change is zero - remove from tracking
                this.changes.delete(path);
                this.notify();
            } else {
                // Update with merged state
                this.changes.set(path, { path, before: originalBefore, after, metadata: mergedMetadata });
                this.notify(change);
            }
        } else {
            // New change
            this.changes.set(path, { path, before, after, metadata });
            this.notify(change);
        }
    }

    getChanges()      // Returns Array of all changes
    getChange(path)   // Get specific change
    hasChange(path)   // Check if path has pending changes

    // Accept: Remove from tracker (keep file as-is)
    accept(path) {
        this.changes.delete(path);
        this.notify(change);
    }
    acceptAll() {
        this.changes.clear();
        this.notify();
    }

    // Reject: REVERT file to original state, then remove from tracker
    async reject(path) {
        const change = this.changes.get(path);
        if (!change) return;
        try {
            if (change.before === undefined && change.after !== undefined) {
                // File was CREATED -> DELETE it
                await unlink(path);
            } else if (change.before !== undefined && change.after === undefined) {
                // File was DELETED -> RECREATE it
                await writeText(path, change.before, workspacePath);
            } else if (change.before !== undefined && change.after !== undefined) {
                // File was MODIFIED -> RESTORE original content
                await writeText(path, change.before, workspacePath);
            }
        } catch { /* Silently ignore revert errors */ }
        finally {
            this.changes.delete(path);
            this.notify(change);
        }
    }

    async rejectAll() {
        const paths = Array.from(this.changes.keys());
        await Promise.all(paths.map(path => this.reject(path)));
    }

    clear() {
        this.changes.clear();  // No revert, just clear tracking
    }
}

// Cursor uses the standard `diff` npm package (v8.0.2) for computing diffs.
// This implements the Myers O(ND) diff algorithm with optimizations.

class Diff {
    diff(oldStr, newStr, options = {}) {
        const oldString = this.castInput(oldStr, options);
        const newString = this.castInput(newStr, options);
        const oldTokens = this.removeEmpty(this.tokenize(oldString, options));
        const newTokens = this.removeEmpty(this.tokenize(newString, options));
        return this.diffWithOptionsObj(oldTokens, newTokens, options, callback);
    }

    diffWithOptionsObj(oldTokens, newTokens, options, callback) {
        const newLen = newTokens.length, oldLen = oldTokens.length;
        let editLength = 1;
        let maxEditLength = newLen + oldLen;
        if (options.maxEditLength != null) {
            maxEditLength = Math.min(maxEditLength, options.maxEditLength);
        }
        const maxExecutionTime = options.timeout ?? Infinity;
        const abortAfterTimestamp = Date.now() + maxExecutionTime;

        const bestPath = [{ oldPos: -1, lastComponent: undefined }];

        // Seed editLength = 0 (content starts with same values)
        let newPos = this.extractCommon(bestPath[0], newTokens, oldTokens, 0, options);
        if (bestPath[0].oldPos + 1 >= oldLen && newPos + 1 >= newLen) {
            return done(this.buildValues(bestPath[0].lastComponent, newTokens, oldTokens));
        }

        // Optimization: track min/max diagonal to prune search space
        let minDiagonalToConsider = -Infinity, maxDiagonalToConsider = Infinity;

        const execEditLength = () => {
            for (let diagonalPath = Math.max(minDiagonalToConsider, -editLength);
                 diagonalPath <= Math.min(maxDiagonalToConsider, editLength);
                 diagonalPath += 2) {

                let basePath;
                const removePath = bestPath[diagonalPath - 1];
                const addPath = bestPath[diagonalPath + 1];

                let canAdd = addPath && (addPath.oldPos - diagonalPath) >= 0
                             && (addPath.oldPos - diagonalPath) < newLen;
                const canRemove = removePath && removePath.oldPos + 1 < oldLen;

                if (!canAdd && !canRemove) {
                    bestPath[diagonalPath] = undefined;
                    continue;
                }

                // Select diagonal farthest from origin
                if (!canRemove || (canAdd && removePath.oldPos < addPath.oldPos)) {
                    basePath = this.addToPath(addPath, true, false, 0, options);
                } else {
                    basePath = this.addToPath(removePath, false, true, 1, options);
                }

                newPos = this.extractCommon(basePath, newTokens, oldTokens, diagonalPath, options);

                if (basePath.oldPos + 1 >= oldLen && newPos + 1 >= newLen) {
                    return done(this.buildValues(basePath.lastComponent, newTokens, oldTokens));
                } else {
                    bestPath[diagonalPath] = basePath;
                    // Prune diagonals when hitting edges
                    if (basePath.oldPos + 1 >= oldLen)
                        maxDiagonalToConsider = Math.min(maxDiagonalToConsider, diagonalPath - 1);
                    if (newPos + 1 >= newLen)
                        minDiagonalToConsider = Math.max(minDiagonalToConsider, diagonalPath + 1);
                }
            }
            editLength++;
        };

        // Sync loop with timeout
        while (editLength <= maxEditLength && Date.now() <= abortAfterTimestamp) {
            const ret = execEditLength();
            if (ret) return ret;
        }
    }

    extractCommon(basePath, newTokens, oldTokens, diagonalPath, options) {
        let oldPos = basePath.oldPos;
        let newPos = oldPos - diagonalPath;
        let commonCount = 0;
        while (newPos + 1 < newLen && oldPos + 1 < oldLen &&
               this.equals(oldTokens[oldPos + 1], newTokens[newPos + 1], options)) {
            newPos++; oldPos++; commonCount++;
        }
        if (commonCount) {
            basePath.lastComponent = { count: commonCount, previousComponent: basePath.lastComponent,
                                       added: false, removed: false };
        }
        basePath.oldPos = oldPos;
        return newPos;
    }

    equals(left, right, options) {
        if (options.comparator) return options.comparator(left, right);
        return left === right || (!!options.ignoreCase && left.toLowerCase() === right.toLowerCase());
    }
}

function structuredPatch(oldFileName, newFileName, oldStr, newStr,
                         oldHeader, newHeader, options) {
    if (options.context === undefined) options.context = 4;

    // Uses diffLines() to compute line-level diff
    const diff = diffLines(oldStr, newStr, options);

    // Build hunks from diff results
    const hunks = [];
    let oldRangeStart = 0, newRangeStart = 0, curRange = [];
    let oldLine = 1, newLine = 1;

    for (let i = 0; i < diff.length; i++) {
        const current = diff[i];
        const lines = current.lines || splitLines(current.value);

        if (current.added || current.removed) {
            // Track range start
            if (!oldRangeStart) {
                oldRangeStart = oldLine;
                newRangeStart = newLine;
                // Include context lines from previous chunk
                curRange = context > 0 ? contextLines(prev.lines.slice(-context)) : [];
            }
            // Add +/- lines
            for (const line of lines) {
                curRange.push((current.added ? '+' : '-') + line);
            }
        } else {
            // Context lines - close hunk or continue
            if (oldRangeStart) {
                if (lines.length <= context * 2 && i < diff.length - 2) {
                    // Overlapping - merge hunks
                } else {
                    // Close hunk
                    hunks.push({ oldStart, oldLines, newStart, newLines, lines: curRange });
                }
            }
        }
    }

    return { oldFileName, newFileName, oldHeader, newHeader, hunks };
}

// Size threshold: 256KB (matches rest of codebase)
const DIFF_SIZE_THRESHOLD_BYTES = 256 * 1024;

// Compute edit hunks efficiently (only changed portions, not full content)
function computeEditHunks(oldContent, newContent) {
    const patch = structuredPatch("", "", oldContent, newContent, "", "", {
        context: 0,  // No context lines - just the changes
    });

    if (patch.hunks.length === 0) return [];

    return patch.hunks.map(hunk => {
        const oldLines = [];
        const newLines = [];
        for (const line of hunk.lines) {
            if (line.startsWith("-")) oldLines.push(line.slice(1));
            else if (line.startsWith("+")) newLines.push(line.slice(1));
        }
        return {
            old_string: oldLines.join("\n"),
            new_string: newLines.join("\n"),
        };
    });
}

                edits = [{ old_string: "", new_string: args.fileText }];
            } else if (contentBeforeWrite === undefined) {
                // New file
                edits = [{ old_string: "", new_string: args.fileText }];
            } else if (contentBeforeWrite === args.fileText) {
                // No changes
                edits = [];
            } else {
                // Compute diff hunks
                edits = computeEditHunks(contentBeforeWrite, args.fileText);
            }

            // Fire afterFileEdit hook with computed edits
            await hookExecutor.executeHookForStep(
                HookStep.afterFileEdit,
                { ...baseHookRequest, file_path: args.path, edits }
            );
        },

        // Cleanup stored pre-write state
        runCleanup: (toolUseId) => preWriteStateMap.delete(toolUseId),
    };
}

-- Reading text files --
async function readText(file) {
    const buf = await readFile(file);
    const format = getFormatForBuffer(buf);
    if (format.isBinaryFile) throw new Error("Binary file cannot be decoded as text");

    const encodingToUse = cachedEncoding || format.encoding;
    const { text, encoding } = decodeBufferWithEncoding(buf, encodingToUse);
    const str = stripUtf8Bom(text);

    // NORMALIZE: CRLF -> LF on read
    if (format.lineEnding === LineEnding.CRLF) {
        return str.replaceAll("\r\n", "\n");
    }
    return str;
}

const getIsNewFileAndDiffString = async () => {
    let isNewFile = false;
    let originalContents = "";
    try {
        originalContents = await readText(absolutePath);
    } catch { isNewFile = true; }

    let diffString = "";
    if (isNewFile) {
        diffString = newContents;
    } else {
        const ensureNL = (s) => (s.endsWith("\n") ? s : `${s}\n`);
        const { hunks } = structuredPatch("a", "b",
            ensureNL(originalContents), ensureNL(newContents), "", "", { context: 2 });
        diffString = hunks
            .map(h => h.lines
                .map(line => line.length > 0 ? `${line[0]} ${line.slice(1)}` : line)
                .join("\n"))
            .join("\n...\n");
    }
    return { isNewFile, diffString };
};

================================================================================
SUMMARY: WHAT THE LLM OUTPUTS vs WHAT GETS APPLIED
================================================================================

The LLM outputs two types of edit instructions:

1. **EditFile (search-and-replace)**: The LLM outputs:
   - `relative_workspace_path`: Target file
   - `old_string`: Text to find in the file
   - `new_string`: Replacement text
   - Optional: `allow_multiple_matches`, `use_whitespace_insensitive_fallback`,
     `use_did_you_mean_fuzzy_match`, `line_ranges`

   The Cursor IDE/aiserver resolves the search-replace match. The exec-daemon
   receives the FINAL resolved `fileText` and writes it via LocalWriteExecutor.
   The search-replace matching algorithm (including whitespace-insensitive
   fallback and fuzzy matching) is NOT in the exec-daemon bundle.

2. **ApplyAgentDiff**: The LLM outputs:
   - `agent_id`: Which agent's accumulated changes to apply
   The server resolves all pending changes and returns per-file
   AppliedAgentChange entries (created/modified/deleted with before/after).

3. **EditFile (speculative/full content)**: The LLM outputs:
   - `contents`: Full new file contents
   - `instructions`: What to change
   The Cursor IDE applies a speculative edit model to generate the diff.

The exec-daemon's diff computation (Myers algorithm via `diff@8.0.2`) is used
ONLY for:
- Computing unified diff strings for approval dialogs
- Computing edit hunks for afterFileEdit hooks
- NOT for applying edits (that happens in the IDE or server)
